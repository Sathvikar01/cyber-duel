# Cyberpunk Duel AI

A 3D AI-powered cyberpunk fighting game built with Gradio + FastAPI.
The AI opponent uses a fine-tuned Gemma 3 model to counter your moves in real time.

## Setup
Add your `HF_TOKEN` as a Secret in Space settings to enable AI inference.
